# Proyecto de Login y Carrito de Compras

## Descripción
Este fue mi primer proyecto de Python donde fusioné un sistema de **Login de Usuario** con un **Sistema de Carrito de Compras**.

## Características
*   Sistema de autenticación básico.
*   Funcionalidad para añadir/remover ítems del carrito y pagar.
*   Funcionalidad para cuenta de administrador de editar los productos del catalogo
*   A prueba de errores (uso de Try/Except)

## Instalación y Ejecución
Para correr este proyecto, necesitas tener Python instalado.

1.  Clona el repositorio:
    ```bash
    git clone [URL_DEL_REPO]
    ```
2.  Navega al directorio:
    ```bash
    cd ProyectoFusion
    ```
3.  Ejecuta el script principal:
    ```bash
    python login_system.py
    ```

## Tecnologías Usadas
*   Python
