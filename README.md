# Login and Shopping Cart Project

## Description
This was my first Python project where I merged a **User Login** system with a **Shopping Cart System**.

## Features
* Basic authentication system.
* Functionality to add/remove items from the cart and pay.
* Functionality for administrator account to edit catalog products
* Error proofing (use of Try/Except)

## Installation and Execution
To run this project, you need to have Python installed.

1. Clone the repository: 
```bash 
git clone [URL_DEL_REPO] 
```
2. Navigate to the directory: 
```bash 
ProjectFusion cd 
```
3. Run the main script: 
```bash 
python login_system.py 
```
